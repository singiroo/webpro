1. circle객체 생성

- 속성 : 반지름(rad)
- 메소드 : 면적 구하기(getArea()), 둘레구하기(getCircum())



2. Rect객체 생성

- 속성 : 가로길이(width), 세로길이(height), 이름(name)
- 메소드 : 면적 구하기(getArea()), 둘레구하기(getCircum())



3. addr객체

- var addr={ }
- 속성 : 정해진 속성이 없다, 추후에(확인 버튼을 클릭하면) 동적으로 추가해서 사용할 수 있다.
- 메소드 : 정해진 메소드가 없다, 후에(확인 버튼을 클릭하면) 동적으로 추가해서 사용할 수 있다.
- addr.name = "홍길동", ...
